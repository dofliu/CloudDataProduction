# 對外連線(校外學生連得到)

課程是「回家作業」型:學生在校外要能連到常駐 5090 主機。分兩條路,對應兩種流量:

| 流量 | 用途 | 通道 | 對外形式 |
|------|------|------|----------|
| **HTTP / WebSocket** | 前端世界 / 目錄 / 儀表板 + `/api` + `/ws` | **Cloudflare Tunnel** | `https://clouddata.example.com` |
| **原生工業協定** | 學生自寫 client 連 Modbus / OPC-UA / MQTT | **Tailscale**(mesh) | `mendon-5090:6020 / 6041 / 6083` |

原則:**HTTP 走 Cloudflare(免公開 IP、免開防火牆);裸 TCP 協定走 Tailscale mesh**(Cloudflare Tunnel 不轉裸 TCP)。教師控制埠一律不對外。

---

## 一、HTTP / WebSocket:Cloudflare Tunnel

### 1. 後端同源提供前端(單一 tunnel 就夠)

先把前端建置,並讓後端同源提供它 —— 這樣「一條 tunnel → :8077」就同時涵蓋網頁 + API + WS,不必再架另一個網頁伺服器。

```powershell
cd web; npm install; npm run build          # 產生 web/dist
# 啟動後端時設 WEB_DIST 指向 dist(見 .env)
```

`.env`:
```
WEB_DIST=web/dist
```

啟動後日誌會出現 `[api] 同源提供前端靜態檔:web/dist`。本機驗證:瀏覽器開 `http://127.0.0.1:8077/` 應直接看到園區世界(而非 JSON);`/api` 仍是 API 資訊索引。

> 不設 `WEB_DIST` 則維持原樣(只有 API);開發期照常用 `npm run dev`(Vite proxy 到 8077)。

### 2. 建 tunnel 並常駐

```powershell
cloudflared tunnel login
cloudflared tunnel create clouddata           # 得 <TUNNEL_ID> 與憑證 json
# 編 deploy/cloudflared/config.yml:填 <TUNNEL_ID>、credentials-file、hostname
cloudflared tunnel route dns clouddata clouddata.example.com
cloudflared tunnel --config deploy\cloudflared\config.yml run
# 常駐(開機自動):
cloudflared service install --config D:\...\deploy\cloudflared\config.yml
```

設定範本見 [`deploy/cloudflared/config.yml`](../deploy/cloudflared/config.yml)。WebSocket(`/ws/telemetry`、`/ws/events`)走 cloudflared 沒有額外設定。

### 3. 安全

- Tunnel 只映射 `:8077`;**教師控制埠(`MODBUS_CONTROL_PORT`)絕不對外**。
- 教師面 API 端點靠 `TEACHER_TOKEN` 保護(`.env` 設一個強 token)。
- 可在 Cloudflare Zero Trust 對 `clouddata.example.com` 加 Access 政策(限學校網域 Email / 學生群組)當第二道門。

## 二、原生協定:Tailscale

學生要用自己的 Python client 連 Modbus/OPC-UA/MQTT(裸 TCP),用 Tailscale 把 5090 主機與學生機器組進同一個 mesh。

### 1. 主機與學生端安裝

```powershell
# 5090 主機
tailscale up --hostname mendon-5090
# 學生各自:安裝 Tailscale、登入同一個 tailnet(用學校帳號 / 受邀)
```

### 2. 設備目錄公布正確位址

設備目錄(`/api/catalog`)公布給學生連協定的 host。設成主機的 Tailscale 名稱,學生抄到的連線資訊才連得到:

`.env`:
```
PUBLIC_HOST=mendon-5090        # 或 100.x.x.x 的 Tailscale IP
```

學生連線範例:`ModbusTcpClient("mendon-5090", port=6020)`。

### 3. Tailscale ACL(限學生群組 + 只開這幾個埠)

在 Tailscale 管理台的 ACL(範例):

```jsonc
{
  "groups": { "group:students": ["stu1@school.edu", "stu2@school.edu"] },
  "acls": [
    // 學生只能連 5090 主機的協定埠(唯讀遙測),不能碰其他埠 / 主機
    { "action": "accept", "src": ["group:students"],
      "dst": ["mendon-5090:6020,6041,6083,6100-6144"] }
  ]
}
```

- `6020` Modbus / `6041` OPC-UA / `6083` MQTT(+ `6100-6144` multiport 進階)。
- **不要**把 `MODBUS_CONTROL_PORT`(教師寫線圈)放進學生可達埠。
